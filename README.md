## Protótipo MABEL PTQA

## Descrição
Projeto desenvolvido durante o curso técnico em Desenvolvimento de Sistemas no IFSC.
O objetivo do projeto é desenvolver uma interface de monitoramento de um dispositivo de medição da qualidade do ar (ptqa) e também de um dispositivo para monitorar uma colmeia de abelhas (Mabel). O principal objetivo foi aprendizado, tendo em vista que foi um trabalho acadêmico.

## Tecnologias Utilizadas
- PHP
- HTML / CSS / JavaScript
- MySQL
- XAMPP
- Chart.js
- Git / GitHub

## Funcionalidades
- Monitoramento
- Análise de dados

## Como executar?
Copie o repositório no seu LocalHost, acesse pela web, abra o arquivo index.html.
